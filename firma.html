Cómo instalar la firma

  1. Selecciona todo el contenido dentro del recuadro blanco de abajo (Ctrl+A dentro del área o arrástralo con el cursor)
  2. Cópialo (Ctrl+C)
  3. En Gmail: Configuración → Ver toda la configuración → General → Firma → Pegar
  4. En Outlook: Archivo → Opciones → Correo → Firmas → Nueva → Pegar
  5. Guarda los cambios
